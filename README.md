## 评估能力说明

本工具提供了一个统一的评估框架，用于对多种格公钥密码算法进行性能分析，主要包括以下三个方面：

### 1. 公钥、私钥与密文尺寸评估

本工具能够在给定参数集的情况下，对不同格密码方案的：

- 公钥尺寸
- 私钥尺寸
- 密文尺寸

进行理论估算。尺寸计算基于多项式、向量和矩阵在模环上的数学表示形式，能够反映不同参数选择下的存储开销。

目前支持的算法类型包括：

- LWE / LWR
- RLWE / RLWR
- MLWE / MLWR
- NTRU 及其变体

---

### 2. 加解密计算复杂度评估

本工具能够对以下阶段的计算复杂度进行估算：

- 密钥生成
- 加密过程
- 解密过程

评估方法基于主要代数运算的数量统计，包括：

- 模加法与模乘法
- 多项式乘法（NTT次数）
- 向量-矩阵乘法

所得结果为理论复杂度估计，可用于不同算法及参数集之间的**相对性能比较**，而非具体实现平台上的实际运行时间。

---

### 3. 解密失败概率评估

针对包含噪声项的格密码方案，本工具构建了最终解密误差的概率模型，通过以下步骤计算解密失败概率：

- 建立各噪声项的离散概率分布
- 进行多重离散卷积
- 计算超过解密阈值的尾部概率

目前已实现对以下类型方案的解密失败概率建模：

- LWE / RLWE
- MLWE / MLWR
- NTRU 类结构

输出结果通常以如下形式给出：解密失败概率 ≈ 2^-k

该表示方式是格密码参数选型中常用的安全指标。

---

### 总结

综上，本工具从三个维度对格公钥密码算法进行系统评估：

| 维度   | 含义                       |
| ------ | -------------------------- |
| 尺寸   | 公钥、私钥和密文的存储开销 |
| 时间   | 加解密过程的理论计算复杂度 |
| 可靠性 | 解密失败的概率             |

该工具可用于在统一模型下，对不同格密码方案进行参数设计层面的对比分析，为算法选型和参数设置提供参考依据。
